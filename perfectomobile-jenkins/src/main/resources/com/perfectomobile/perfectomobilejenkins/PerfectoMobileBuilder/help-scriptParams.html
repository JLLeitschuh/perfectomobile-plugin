Click the Parameters List button to display the runtime parameters defined in the script.
Fill in the parameter values.

String/integer: simply type them
For example, “appIdentifier(string)=Perfecto Mobile OSE”

Media/Data table files: will be uploaded automatically to the MobileCloud repository.
Type in the repository path where to upload and the path to the file or artifact to copy, separated by a semicolon.
Format, Repository path;local path
For example, “myApp(media)=PRIVATE:TestApp.apk;c:\mybuild\TestApp.apk”

Handsets: click on Device List to see the available devices.
For example, “DUT(handset)=TA8830NFLG”
